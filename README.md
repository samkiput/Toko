3.lakukan inner join
  <img width="1280" height="201" alt="image" src="https://github.com/user-attachments/assets/edf84e22-5125-44c4-9962-3bd865f18fa7" />
  Inner Join adalah jenis join yang hanya menampilkan data yang cocok di kedua tabel.
  Ini dipakai saat kamu hanya ingin melihat baris yang punya match di kedua tabel.
Kalau satu data di salah satu tabel tidak punya pasangan di tabel lain tidak muncul.


4.lakukan left join 
  <img width="1280" height="239" alt="image" src="https://github.com/user-attachments/assets/b7cf641e-dec9-452f-bc99-d8e391272732" />

Left Join (atau LEFT OUTER JOIN) akan menampilkan semua data dari tabel kiri + data dari tabel kanan jika cocok.

Kalau di tabel kanan tidak punya pasangan ditulis NULL di hasilnya.
  


5. lakukan right join
   <img width="1280" height="254" alt="image" src="https://github.com/user-attachments/assets/dbbe115d-fea2-4630-9872-f743b8fbfcda" />
    Right Join (atau RIGHT OUTER JOIN) kebalikan dari LEFT JOIN:
Menampilkan semua data dari tabel kanan, plus data dari tabel kiri jika cocok.

Kalau di tabel kiri tidak punya pasangan → kolom dari kiri jadi NULL.
